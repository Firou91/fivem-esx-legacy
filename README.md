- Start `Apache` & `MySQL`
- Open `MySQL` & select `Import` ;
- Choose the *fESX.sql* file & click `Import`.

## Hosting :

### 1 - Base
- `Download ZIP` ;
- Extract all folders ;
- Drop all files on your server folder ;
- Configure your *server.cfg*.
### 2 - SQL
- Go to `PhpMyAdmin` or `HeidiSQL` ;
- Select `Import` ;
- Choose the *fESX.sql* file & click `Import`.
